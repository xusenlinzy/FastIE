此项目为开源【文本分类】、【实体抽取】、【关系抽取】和【事件抽取】模型的训练和推理提供统一的框架，具有以下特性


+ ✨ 集成多种开源文本分类、实体抽取、关系抽取和事件抽取模型


+ 👑 支持百度 [UIE](https://github.com/PaddlePaddle/PaddleNLP) 模型的训练和推理


+ 🚀 统一的训练和推理框架


+ 🎯 集成对抗训练方法，简便易用
